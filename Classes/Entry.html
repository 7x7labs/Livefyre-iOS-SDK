Entry.h

visibility

The visibility of this Entry.

@property (nonatomic, readonly) enum ContentVisibility visibility

Discussion

The visibility of this Entry.

ContentVisibilityOwner indicates that the Entry can only be seen by the posting user. This normally should not be revealed in the UI.

ContentTypeGroup indicates that the Entry is currently waiting for moderation and so it only visible to the posting user and any moderators for the Livefyre collection.

Posts which cannot be seen by the current user are automatically filtered out, so ContentVisibilityNone will normally not be seen.
